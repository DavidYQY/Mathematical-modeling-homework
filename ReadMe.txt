this is a repository for homework in mathematical modeling course at Tsinghua University.
completed by yqy, Dec,2015
the repository consists of two main task:
1\ a small task, it is a Himcm test about forest fire, maybe in the year of 2001
2\ a big task, contributed by 4 members, about an evacuating model in the stadium. We took stadium in Tsinghua University for example.

To run the project, just copy the file to the local path.
And run:simplified model, which is the main function.

All the simulator is completed by Matlab R2015a
