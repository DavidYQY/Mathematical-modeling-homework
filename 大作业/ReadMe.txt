随着经济和城市化的发展，群众的生活日益丰富，各种大型集会的举办日益频繁。但其中的一个后果是加剧了人群的聚集，导致近年来群体性踩踏事件频发。因此，在人群集中的场所，一旦发生各种紧急事件，如何快速疏散到安全位置，成为一个刻不容缓的热点研究话题。
在紧急情况下，人群不会保证纯粹理性，而会产生特殊的应急心理因素，如从众心理、小团体心理等。这些心理因素极有可能对疏散效率造成重大影响，甚至导致踩踏致使人员伤亡。因此，紧急疏散中必须提前考虑人群的应急心理因素。
现有紧急疏散模型普遍无差异化同等对待个体行为，忽略了不同心理因素对个体乃至对疏散后果的影响。针对此不足，本文以研究紧急疏散中人员的应急心理因素为主要目标。本文思路创新在于：量化心理影响，将数学建模这一相对理性的工具与人类感性的心理行为进行结合，建立了一个新颖的仿真模型，该模型物质与精神并重，理性与感性齐鸣，更真实精细化地模拟现实紧急情况下的疏散过程；本文方法创新在于：调研分析了大学校园集会的人员组成，把集会分为五种典型的类型，这项工作属于我们的首创。进一步地，以清华最大室内体育场馆――清华大学综合体育馆的具体布局为例，利用我们的仿真模型的计算，动态模拟了整个过程得到了针对不同类型的集会的安全建议。内容具体，可操作性强，这些建议可以提供给学校管理部门，并切实地投入使用。
本文主要做了一下研究工作：
（1）模型建立。基于Agent模型和元胞自动机的原理，建立紧急疏散情况下的数学模型，包括基础的个体速度模型和个体方向选择模型，模型中还重点考虑和量化了三种心理因素，包括小团体心理、从众心理和逆流心理。
（2）集会类型分类。通过实地调研、收集资料和整理分析，得到清华大学师生的人员构成及各类集会的特点，提炼出五种不同类型综合体育馆集会，包括开学毕业典礼、文艺活动、体育赛事、周年庆典和人才招聘会。
（3）模型的一般化验证。针对总体情况进行一般化的仿真模拟，初步验证了我们模型的合理性和可行性。
（4）模型的实际应用。选取了综合体育馆举办集会中毕业典礼和人才招聘会两种具有代表性的集会，根据特点调整模型参数，进行仿真模拟，根据仿真结果，提出针对性建议。具体包括：针对毕业典礼，应在东1、西1出口处设置疏导人员，将人群引向北1、北2出口；针对人才招聘会，则应在南1、南2出口队伍队尾增设疏导人员，将队列后部人群引向北1、北2出口。根据对比模拟发现，实行建议后，疏散时间分别同比下降39%和28%，验证了模型的有效性。
（5）提出安全指南。根据所做实验，结合现实管理现状和不足，总结出一个值得重视的安全指南。
